Project Overview

This project aims to classify human physical activities as Walking or Running using machine learning algorithms. The model is trained on sensor data collected from mobile devices, enabling automatic activity recognition based on movement patterns.

Dataset

The dataset contains accelerometer and gyroscope sensor readings along with activity labels. These features help identify whether a person is walking or running.

Objectives
Analyze sensor data through Exploratory Data Analysis (EDA)
Build and compare machine learning models
Predict human activities accurately
Evaluate model performance using classification metrics
